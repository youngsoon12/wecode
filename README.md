## 1. git init

현재 디렉토리를 로컬저장소로 설정한다. (init을 취소하려면 rm -r .git)

## 2. git remote add <로컬저장소 이름> <'github repository url'>

연동시킬 깃허브의 url을 입력해주어 로컬저장소와 연결 시킨다.

## 3. git add

파일을 준비영역으로 옮긴다 (add뒤에 .을 붙일 시 워킹 디렉터리 내 모든 파일츨 추가)

## 4. git commit

준비영역의 파일을 로컬 저장소에 저장한다.

## 5. git push

로컬 저장소에 commit 된 파일들이 github 벤치로 저장된다.
